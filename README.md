
# BSPWM Dotfiles

Dotfiles for my BSPWM setup




## Configuation

- **Window Manager 🎨 :** BSPWM 
- **Terminal Emulator 🐱 :** Kitty, ST
- **Panel 🍧 :** Polybar 
- **Launcher 🚀 :** Rofi
- **File Manager 📂 :** PCManFM 
- **Notification 🔔 :** Dunst
- **Editor :** NeoVIM Custom
- **Fetch :** Neofetch 
- **Compositor ✨ :** Picom-Ibhagwan 
- **Wallpaper 🖼️ :** Nitrogen
- **Shell 🐚 :** Fish
- **OS :** LegionOS (Arch Linux)
